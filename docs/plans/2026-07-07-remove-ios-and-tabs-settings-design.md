# Дизайн-документ: Удаление настроек iOS-стиля и Режима вкладок из профиля

## Проблема / Контекст
В разделе редактирования настроек профиля (компонент настроек `ProfileSettingsModal`) присутствуют переключатели «Стиль iOS 📱» и «Режим вкладок 🧭». По требованию пользователя эти опции необходимо убрать из интерфейса настроек профиля, оставив только настройку двухфакторной аутентификации (MFA).

## Описание изменений

### 1. Изменения в UI компонентах

#### `src/features/Profile/ui/ProfileSettingsModal.tsx`
* Удаляем из props-интерфейса `IProps` параметры настройки `navbarVariant`, `setNavbarVariant`, `tabMode` и `setTabMode`.
* Из JSX-кода удаляем соответствующие блоки переключателей (Switch) и контейнеры.
* Внутри разметки оставляем только компонент `<MfaSecurity />`.
* В соответствии с системным манифестом разработки (`RULE[GEMINI.md]`) полностью удаляем все комментарии из файла.

#### `src/features/Profile/Profile.tsx`
* Удаляем неиспользуемые импорты хуков `useNavbar` и `useTabs`.
* Удаляем вызовы хуков `useNavbar()` и `useTabs()` в теле компонента `Profile`.
* Удаляем передачу удалённых пропсов в `<ProfileSettingsModal>`.
* В соответствии с системным манифестом разработки (`RULE[GEMINI.md]`) полностью удаляем все комментарии из файла.
