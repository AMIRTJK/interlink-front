# Дизайн-документ: Добавление блока «Биография» в профиль

## Проблема / Контекст
В профиле пользователя на вкладке информации (`ProfileInfoTab.tsx`) необходимо заменить блок «Роли и доступ» на блок «Биография». В новом блоке должна отображаться информация из поля `user.bio` (берется из `GET /api/v1/auth/me`).

## Описание изменений

### 1. Изменение отображаемых блоков в `ProfileInfoTab.tsx`
* Заменяем блок `<Card title="Роли и доступ">` на `<Card title="Биография">`.
* Внутри новой карточки выводим текст биографии пользователя из `userData?.bio`.
* Если биография отсутствует или равна `null`, отображаем текст-заглушку «Не указано» курсивом (стиль: `text-zinc-400 dark:text-zinc-500 italic`).
* Разметку биографии оборачиваем в `div` с классом `col-span-full` для занятия полной ширины сетки.
* Удаляем импорт неиспользуемой иконки `ShieldCheck` из `lucide-react` для предотвращения ошибок сборки/линтера.
* В соответствии с `RULE[GEMINI.md]` комментарии отсутствуют.
