# Дизайн-документ: Вкладка «Мои файлы» в Профиле

**Дата:** 2026-07-02
**Тема:** Создание интерактивной вкладки «Мои файлы» в соответствии с предоставленным макетом.

## 📌 Цели и требования
- Создать премиум-интерфейс для управления личными файлами пользователя.
- Соответствовать приложенному дизайну: градиенты, карточки, прогресс-бары, иконки.
- Обеспечить адаптивную и интерактивную верстку (поддержка Grid/List видов, поиск, фильтрация по категориям, закрепление файлов).

## 🎨 Визуальные и Функциональные Компоненты

1. **Шапка вкладки:**
   - Название «Мои файлы» и счетчик общего количества файлов.
   - Поле поиска по названию.
   - Сортировка по дате (выпадающий список).
   - Переключатель отображения (сетка / список).
   - Кнопка «Загрузить» с оранжево-пурпурным градиентом.

2. **Категории (Фильтры):**
   - Пилюли фильтров: «Все файлы», «Рабочие», «Документы».
   - Кнопка «+ Новая» для добавления новых категорий.

3. **Закрепленные файлы:**
   - Секция «ЗАКРЕПЛЁННЫЕ» с чипсах файлов (например, `Отчёт_2025_Q1.pdf`, `База_данных_v2.xlsx`).
   - Кнопка открепления (крестик).

4. **Зона загрузки (Drag & Drop):**
   - Блок с пунктирной рамкой: «Перетащите файлы или нажмите, чтобы загрузить».

5. **Сетка файлов (File Grid):**
   - Карточки файлов с цветным превью (в зависимости от типа: архивы — оранжевый градиент, документы — сине-голубой, таблицы — зеленый с сеткой, изображения — миниатюра картинки, PDF — красный с надписью "PDF").
   - Имя файла, размер и дата изменения.
   - Интерактивный индикатор закрепления (булавка).

6. **Панель хранилища (Storage Bar):**
   - Индикатор заполнения хранилища (градиентный прогресс-бар: оранжево-розово-фиолетовый).
   - Легенда с разделением по типам файлов и их количеством.

## 🛠️ Стек технологий
- **Фреймворк:** React (TypeScript)
- **Стилизация:** Tailwind CSS + локальный `style.css` (при необходимости)
- **Анимации:** `framer-motion` для плавного переключения видов, фильтров и модалок
- **Иконки:** `lucide-react` (или Ant Design при необходимости)

## 📁 Структура папки компонента
В соответствии с манифестом GEMINI.md:

- `src/features/Profile/ui/tabs/FilesTab.css` — дополнительные стили для сложных градиентов или сетки (если не покрывается Tailwind).
- `src/features/Profile/ui/tabs/mockData.ts` — исходные моки файлов.
