# Инструкция по загрузке паспорта

Здесь лежит изображение-инструкция, которое показывается в модальном окне
**«Новый сотрудник»** (шаг загрузки паспорта, `src/features/Hr/ui/PassportUploadStep.tsx`).

## Как заменить картинку

Просто перезапишите файл **`passport-upload-guide.png`** своим реальным
изображением инструкции (с примерами правильной и неправильной загрузки).

- Имя файла менять не нужно — компонент ссылается на него по константе
  `GUIDE_IMAGE_SRC` в `PassportUploadStep.tsx`.
- Если хотите другое имя/формат (`.png`, `.webp`) — поменяйте одну строку
  `GUIDE_IMAGE_SRC` в `PassportUploadStep.tsx`.
- Рекомендуется вертикальное изображение хорошего качества: пользователь сможет
  открыть его на весь экран, приблизить (zoom) и перемещать (pan).

Текущий файл — **временная заглушка**, сгенерированная автоматически.
